# Dolphin Gang Tour 使用指南

## 1. 安装

| 平台 | 步骤 |
|------|------|
| **Windows** | 下载 `Setup.exe` → 双击安装 → 桌面启动 |
| **macOS** | 下载 `.dmg` → 拖入「应用程序」→ 首次打开若被拦截：系统设置 → 隐私与安全性 → 仍要打开 |
| **Android** | 下载 `.apk` → 卸载旧版（如有）→ 允许未知来源 → 安装 |

Beta 授权码：**`DGT-BETA-2026`**（仅校验版本，不收集设备/NFC 数据）

---

## 2. 首次启动

1. 阅读并同意用户协议与隐私政策  
2. 等待启动动画完成  
3. 可选：设置 → 配置在线模型 API Key（不配置也可使用本地 FAQ 降级）

---

## 3. 无硬件体验（虚拟设备）

1. 左侧侧栏点击 **扫描设备**  
2. 选择 **Flipper Zero (Virtual Demo)**  
3. 可体验：资源导入流程、屏幕镜像、GPIO 沙盘、学习助手

---

## 4. 连接真实 Flipper

1. USB 数据线连接电脑（Android 需 OTG 拓展坞）  
2. 侧栏显示 **CONNECTED** 后可用：  
   - **一键导入** — 选择资源包写入 SD 卡  
   - **固件管理** — 下载并刷写社区固件  
   - **屏幕镜像** — 实时画面 + 虚拟按键  
   - **故障诊断** — 端口/驱动/SD 卡检测

---

## 5. 学习助手

- 侧栏 **学习助手** 进入对话  
- 底部输入问题，或选择 **课程学习** 按章节进行  
- 助手回复标注 **合成内容**（合规标识）

---

## 6. 常见问题

| 问题 | 处理 |
|------|------|
| Android 无法安装 | 使用 [Releases](https://github.com/immaotianyi/dolphin-gang-tour/releases) 最新已签名 APK，先卸载旧版 |
| Android 闪退 | 安装 v2.0beta 最新 APK |
| 找不到设备 | 关闭 qFlipper 等占用端口的程序后重试 |
| Mac 提示无法打开 | 系统设置 → 隐私与安全性 → 仍要打开 |
