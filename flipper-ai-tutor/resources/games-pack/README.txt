============================================================
 Flipper Zero Games Pack
============================================================

Flipper Zero 游戏以 .fap（Flipper Application Package）文件的形式分发。
.fap 是使用 Flipper 固件 SDK 从 C 源码编译出来的二进制文件，无法用
文本编辑器手工创建，因此本包内不包含任何游戏文件。

------------------------------------------------------------
 如何获取游戏（从应用市场下载）
------------------------------------------------------------
1. 打开 Flipper Zero 手机 App（iOS / Android），或访问 Flipper Lab
   应用目录网站：https://lab.flipper.net/apps
2. 进入 "Games" 分类，通过蓝牙 / USB 直接安装到 Flipper；
   或将 .fap 文件下载到 SD 卡的对应目录：
       /ext/apps/Misc
       /ext/apps/Games
3. 在设备上按主菜单右键打开"应用菜单（App Menu）"，即可启动已安装的游戏。

------------------------------------------------------------
 热门 Flipper Zero 游戏（从应用目录安装）
------------------------------------------------------------
- Arkanoid（打砖块）
- 2048
- Game 15（十五数码）

- Pong
- Chess（国际象棋）
- Tic-Tac-Toe（井字棋）
- Doom（Doom 移植）
- Heap Defragmenter / Calibrator（实用兼娱乐）

------------------------------------------------------------
 从源码自行编译游戏（开发者）
------------------------------------------------------------
    git clone https://github.com/flipperdevices/flipperzero-firmware
    cd flipperzero-firmware
    ./fbt fap_<app_name>

编译产物位于 build/ 目录，可拷贝到 SD 卡使用。

------------------------------------------------------------
 说明
------------------------------------------------------------
本包仅作说明用途，未内嵌任何二进制游戏文件。请通过官方 Flipper
应用目录（lab.flipper.net）或可信社区固件（Momentum、Unleashed 等）
的应用商店获取 .fap 游戏文件。
