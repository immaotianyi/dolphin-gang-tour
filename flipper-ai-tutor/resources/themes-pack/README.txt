============================================================
 Flipper Zero Themes Pack
============================================================

Flipper Zero 主题（自定义 UI 图形：图标、动画、字体）并非简单的文本
文件。在自定义固件——尤其是 Momentum 固件——上，主题通过"Asset Pack"
系统分发：它将编译好的资源文件打包成一个二进制归档，由固件在开机时
加载。因此本包内不包含主题二进制文件。

------------------------------------------------------------
 为什么主题无法用纯文本创建
------------------------------------------------------------
- 图标以编译后的 XBM 位图数组形式存储。
- 海豚（dolphin）动画是压缩后的位图帧序列。
- 这些资源由 asset_packer 工具打包成单个归档，供固件资源加载器读取。

------------------------------------------------------------
 如何获取主题（Momentum 固件 Asset Pack）
------------------------------------------------------------
1. 将 Momentum 固件刷入 Flipper Zero（https://momentum-fw.dev）。
2. 打开 Momentum 手机 App 或 Momentum 应用目录。
3. 进入 "Themes" / "Asset Packs" 分类。
4. 安装 Asset Pack，文件会存放在 SD 卡：
       /ext/assets/themes/<theme_name>/
5. 应用主题：Settings > Momentum > Asset Pack > 选择该包，然后重启设备。

------------------------------------------------------------
 自行制作 Asset Pack（进阶）
------------------------------------------------------------
1. git clone https://github.com/Next-Flip/Momentum-Dev
2. 准备图标（PNG -> XBM）与 dolphin 动画帧。
3. 使用仓库中提供的 asset_packer / 打包脚本构建资源归档。
4. 将生成的 Pack 拷贝到 SD 卡 /ext/assets/themes/<your_theme>/，
   并在 Settings > Momentum > Asset Pack 中选择启用。

------------------------------------------------------------
 原厂固件说明
------------------------------------------------------------
原厂（stock）固件不支持通过 Asset Pack 实现完整自定义主题。只能手动
替换 /ext/dolphin/ 下的海豚动画与图标并重新刷写，功能较为有限。

------------------------------------------------------------
 说明
------------------------------------------------------------
本包仅作说明用途，未内嵌任何主题二进制文件。请从 Momentum 固件目录
获取 Asset Pack，或使用 Momentum-Dev 工具链自行构建。
