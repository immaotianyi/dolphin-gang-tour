# 🗺️ Dolphin Gang Tour 项目路线图

> **项目名称**：Dolphin Gang Tour
> **当前版本**：v1.0.0
> **项目定位**：Flipper Zero 全功能桌面伴侣应用
> **技术栈**：Tauri 2.0 + Rust + React + TypeScript

---

## ✅ 已完成 — v1.0.0（2026-07-07）

首个正式版本，核心功能全部落地，覆盖设备连接、AI 教学、资源管理、固件维护、硬件控制等全链路能力。

- 🔌 **设备连接** — USB 自动检测 + 虚拟设备模拟，即插即用
- 🤖 **AI 教学** — 多模型路由 + 流式输出 + 数据脱敏 + 7 节系统化课程
- 📦 **一键导入** — 5 类资源包 + tar 打包 + SHA256 完整性校验
- 🔧 **固件刷写** — 双轨守护（RPC 协议 + dfu-util DFU 救砖模式）
- 📺 **屏幕镜像** — 128×64 实时帧传输 + 物理按键远程遥控
- ⚡ **GPIO 沙盘** — 8 引脚可视化控制 + OTG 开关一键切换
- 🏆 **成就系统 + 电子宠物** — 游戏化激励 + Dolphin 养成玩法
- 🩺 **故障诊断工具** — 一键检测 + 日志收集 + 问题定位
- ⚖️ **法律合规** — 用户协议 + 隐私政策 + BadUSB 安全沙盒
- 🖥️ **多平台 CI** — macOS / Windows / Linux 三端同步构建

---

## 🔄 进行中 — v1.1.0（短期目标，1–2 周内）

聚焦发布工程、生态拓展与体验优化，让应用从「能用」走向「好用」。

- 🚀 **多平台 Release 自动构建** — 基于 GitHub Actions 的全平台自动化发布流水线
- 🪟 **Windows EXE 安装包完善** — 采用 Wix / NSIS 打造标准化安装体验
- 📱 **Android 版本可行性探索** — 调研 USB 通信层重写方案与技术边界
- 🧠 **更多 AI 模型支持** — 本地 Ollama 集成，实现离线私有化大模型
- 🌐 **社区资源包市场** — 内置浏览 / 下载第三方资源包，激活社区生态
- ⬆️ **应用内自动更新** — 接入 Tauri updater，静默升级无需手动下载

---

## 📋 规划中 — v1.2.0 ~ v1.5.0（中期目标，1–3 个月）

深度挖掘 Flipper Zero 硬件能力，构建可扩展的插件化生态，向移动端延伸。

### v1.2.0 — 插件化与 NFC
- 🔌 **插件系统** — 开放第三方扩展接口，支持社区开发者贡献功能模块
- 💳 **NFC 卡模拟 / 克隆可视化** — 图形化操作卡片数据，降低使用门槛

### v1.3.0 — SubGHz 与协作
- 📡 **SubGHz 信号分析与录制** — 频谱可视化 + 信号录制 / 重放 / 编辑
- 👥 **多人协作** — 设备共享与远程协助，跨越地理限制共同调试

### v1.4.0 — 主题与移动端
- 🎨 **主题商店** — 用户自定义 UI 主题，打造个性化桌面伴侣
- 📲 **移动端适配** — Android / iOS 双端版本，随身掌控 Flipper Zero

### v1.5.0 — 国际化
- 🌍 **多语言支持** — 英文、日文等多语种界面，拥抱全球用户

## 🌟 远景构想（长期愿景）

不止于一款工具，更要成为 Flipper Zero 生态的入口与基础设施。

- 🏆 **生态标杆** — 成为 Flipper Zero 生态中最易用、最全面的桌面伴侣
- 🌐 **全平台覆盖** — 桌面 + 移动 + Web 三端联动，随时随地接入
- 🤖 **AI 驱动自动化** — 基于大模型的智能工作流，一键完成复杂操作
- 🌱 **社区驱动生态** — 资源、插件、教程由社区共建共享，正向循环
- 🛠️ **硬件周边配套** — 开发专属配件与扩展模块，软硬件协同进化
