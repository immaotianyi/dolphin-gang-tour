## 快速上手

1. 安装对应平台安装包（见上表）
2. 启动应用，同意用户协议
3. **无 Flipper？** 侧栏扫描 → 选择「虚拟设备」即可体验
4. **有 Flipper？** USB 连接 → 一键导入 / 固件 / 镜像

详细图文说明见 **[docs/USER-GUIDE.md](./docs/USER-GUIDE.md)**
