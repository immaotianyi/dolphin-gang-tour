- **One-click resource import** — IR, NFC, Sub-GHz, BadUSB packs
- **Firmware flashing** — Momentum, Unleashed, OFW, RogueMaster
- **Live screen mirror** — 128×64 with virtual buttons
- **Learning assistant** — step-by-step courses, multi-model chat
- **Virtual device** — full demo flow without hardware
